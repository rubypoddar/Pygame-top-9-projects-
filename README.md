![Solar System Demo](demo.gif)

## About

The 3D Solar System Simulation provides an interactive and educational experience, allowing users to explore the dynamics of our solar system. It offers a realistic representation of planetary orbits and sizes, making it a valuable tool for astronomy enthusiasts and educators alike.

## Features

- **Realistic 3D Representation**: Planets are rendered in three dimensions, providing an immersive experience.
- **Interactive Environment**: Users can navigate the solar system, adjust camera angles, and zoom in/out for detailed exploration.
- **Educational Tool**: Offers insights into celestial dynamics, planet sizes, and orbital mechanics.
- **Customization Options**: Users can tweak various parameters to simulate different scenarios and perspectives.

## Installation

1. Ensure you have Python (3.6+) and Pygame (2.0.1) installed on your system.
2. Clone the repository:

    ```bash
    git clone https://github.com/rubypoddar/Pygame-top-9-projects-.git
    ```

3. Navigate to the project directory:

    ```bash
    cd Pygame-top-9-projects-/3D solar system
    ```

4. Run the simulation:

    ```bash
    python solar_system.py
    ```

## Usage

1. Upon running the simulation, you'll be immersed in a 3D representation of the solar system.
2. Use arrow keys to rotate the camera angle, mouse scroll to zoom in/out, and spacebar to pause/resume the simulation.

# Molecule Simulation

Welcome to the Molecule Simulation! This project simulates the movement and interaction of molecules in a confined space, offering a visually engaging experience of molecular dynamics.

## Description

The Molecule Simulation project provides a virtual laboratory environment where users can observe the behavior of molecules under different conditions. Molecules move freely within the simulation, bouncing off walls and interacting with each other through attraction and repulsion forces. Users can also manipulate the simulation by adding or removing molecules and changing simulation settings.

## Features

- **Dynamic Molecule Movement**: Molecules move within the simulation space, exhibiting realistic behavior such as bouncing off walls and colliding with each other.
- **Interactive Environment**: Users can interact with the simulation by adding new molecules with a left-click, removing molecules with a right-click, and adjusting simulation settings with keyboard shortcuts.
- **Real-Time Visualization**: The simulation provides real-time visualization of molecule movement and interactions, allowing users to observe dynamic patterns and behaviors.
- **Customizable Settings**: Users can customize simulation settings such as molecule color, transparency, and attraction force to create unique simulation environments.

## Requirements

- Python 3.x
- Pygame

## Usage

1. Clone or download the repository to your local machine.
2. Install Pygame by running: `pip install pygame`.
3. Run the simulation by executing the `main.py` file.
4. Interact with the simulation using the following controls:
   - **Left-click**: Add a new molecule at the cursor position.
   - **Right-click**: Remove a molecule near the cursor position.
   - **C**: Change molecule color.
   - **D**: Delete all molecules.
   - **T**: Toggle transparency mode.
5. Observe the movement and interaction of molecules within the simulation space.
